# opendata-ishikawa
日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

石川県が提供する公開データを対話型地図上に表示するWebアプリケーションです。

## デモ
[石川オープンデータマップ](https://code4fukui.github.io/opendata-ishikawa/)

## 機能
- オープンデータを使って、さまざまなタイプの公共施設やインフラストラクチャを表示します
- ユーザーが異なるデータセットを選択し、それらをマップ上で表示できます
- データポイントの周りに円形のゾーンを表示するオプションが提供されています

## 使用法
1. リポジトリをクローンしてください:
```
git clone https://github.com/code4fukui/opendata-ishikawa.git
```
2. `map.html`ファイルをウェブブラウザで開いて、アプリケーションを表示してください。

## データ / API
このアプリケーションは、石川県が提供するオープンデータを使用しています。これらのデータは、[石川県オープンデータカタログ](https://www.pref.ishikawa.lg.jp/opendata/index.html)からアクセスできます。
